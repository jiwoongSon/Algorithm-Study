#  0528 다리를 지나는 트럭

## 풀이
- 트럭은 다리를 FIFO로 통과 --> QUEUE로 생각해보기
    - 시간, 현재 다리 위의 무게를 저장할 정수형 변수
-  que를 만들어서 일단 0으로 초기화 
- 정수형 변수를 선언하여, 대기 중인 트럭의 Index 저장

## 생각해볼 점
- 자바에서 queue를 구현하는 방법들
    - LinkedList
    - **ArrayDeque**
    - PriorityQueue
    - Array 이용하여 함수 직접 구현 등등..
